Setiap kurva memiliki "kepribadian" unikâ€”warna, kecepatan, ketebalan, dan lengkungan yang dihasilkan secara acak (prosedural).

## âœ¨ Fitur Utama

- __Visualisasi Berputar__: Seluruh scene berputar secara global dengan rotasi otomatis.
- __Efek Zoom Dinamis__: Scaling otomatis menggunakan fungsi trigonometri untuk menciptakan efek membesar-mengecil yang halus.
- __Fisika Pantulan__: Setiap kurva bergerak bebas dengan fisika pantulan sederhana saat menyentuh batas layar.
- __Efek Jejak (Trails)__: Menggunakan teknik penumpukan layer semi-transparan untuk menciptakan efek jejak gerakan yang artistik.
- __6 Mode Variasi__: Setiap kurva memiliki salah satu dari 6 pola lengkungan yang berbeda secara acak.
- __200 Kurva__: Memulai dengan 200 kurva yang menciptakan pola visual yang kompleks dan memukau.
- __Kontrol Interaktif__: Pengguna dapat memanipulasi jumlah kurva secara _real-time_ menggunakan keyboard dan mouse.

## ğŸ® Kontrol (Controls)

Anda dapat berinteraksi dengan visualisasi menggunakan tombol berikut:

| Input | Aksi |
| --- | --- |
| __Klik Mouse__ | Menambahkan kurva baru tepat di posisi kursor mouse |
| __`A`__ | Menambah 1 kurva acak |
| __`+`__ | Menambah 5 kurva sekaligus |
| __`D`__ | Menghapus 1 kurva terakhir (_Delete_) |
| __`-`__ | Menghapus 5 kurva sekaligus |
| __`C`__ | Menghapus semua kurva, membersihkan layar (_Clear_) |
| __`R`__ | Reset ulang simulasi ke kondisi awal (200 kurva) |
